# 📊 Matplotlib Data Visualization Assignment

## 📌 Overview
This project demonstrates different types of data visualizations using Python, Pandas, NumPy, and Matplotlib on Kaggle datasets.

The assignment includes the following visualization tasks:

1. Line Plot (Sales Trend)
2. Scatter Plot
3. Bar Plot
4. Multiple Bar Plot
5. Stacked Bar Chart
6. Histogram
7. Pie Chart

---

## 🛠️ Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Jupyter Notebook

---

## 📂 Dataset

Kaggle Datasets Used:

- sales_data_sample.csv
- mobile_sales_data.csv

---

## 📈 Tasks Performed

### Task 1 – Line Plot (Sales Trend)
- Converted numeric months into month names.
- Displayed sales trend over months.
- Added title and axis labels.

---

### Task 2 – Scatter Plot
- Created a scatter plot using numerical columns.
- Displayed the relationship between two numeric variables.
- Added title and labels.

---

### Task 3 – Bar Plot
- Created both Vertical Bar Chart and Horizontal Bar Chart.
- Added title and axis labels.

---

### Task 4 – Multiple Bar Plot
- Compared sales across different years.
- Used proper bar width and alignment.
- Added legend, title and axis labels.

---

### Task 5 – Stacked Bar Chart
- Created stacked bar chart using grouped sales data.
- Added legend and axis labels.

---

### Task 6 – Histogram
- Displayed distribution of numerical data.
- Used appropriate bins.
- Added title and labels.

---

### Task 7 – Pie Chart
- Displayed category-wise market share.
- Added percentage values.
- Added chart title.

## ▶️ How to Run

### 1. Clone the repository

```bash
git clone <repository_link>
```

### 2. Open project folder

```bash
cd Matplotlib-Assignment
```

### 3. Install required libraries

```bash
pip install numpy pandas matplotlib
```

or

```bash
pip install -r requirements.txt
```

### 4. Open Jupyter Notebook

```bash
jupyter notebook
```

Open

```
matplotlib_assignment.ipynb
```

Run all cells sequentially.
